# NoExcuse

**No excuse. Or fail.**

Simple Pomodoro timer with task management. No fluff, just work.

## Features

- 25-minute Pomodoro sessions with breaks
- Task list with subtasks
- Backlog sidebar for future tasks
- Session tracking
    - Green dots for normal hours (up to 16 sessions / 8 hours per day)
    - Red dots for overtime sessions
- Drag & drop to reorder
- Everything stored locally in your browser
